# Introduccion-al-Analisis-de-Datos-y-Microsoft-Excel
Repositorio para la entrega de mi caso práctico de Excel para el Diplomado Análisis y Visualización de Datos 5a. Edición

Este caso practico es muy util para visualizar los datos de una forma dinámica y especifica.
- Dinámica porque con el boton de segmentacion de datos y conectando las tablas dinamicas puedo ver especificamente, y en este caso por ubicacion de sucursal, toda la información que agregué a las tablas dinamicas, como lo son los ingresos por mes, los productos, etc.
Ademas, las graficas tambien cambian y son una manera muy útil de ver y presentar los datos. 
- Especifica porque en la tabla dinamica yo puedo jugar con los datos que quiero ver y con eso obtener la información que necesito.

Al menos 3 puntos claves encontrados provenientes del análisis de datos:
* El mes con mayor ingresos es el mes de junio.
* La categoria más vendida es el Coffee.
* La ubicación de Hell's Kitchen es la que tiene mayor numero de ventas en los meses de enero a junio. 

A mi me ayuda mucho las tablas dinamicas en mi trabajo porque yo manejo una base de datos para una constructora y puedo visualizar y comparar por ejemplo, lo que está estimado por gastar para la obra y en qué se deberá gastar vs. lo que realmente se gastó y si hubo perdida o ganancia, entre otra información valiosa para el buen manejo y control del flujo de dinero y recursos.
